*********************ReadMe before using DigiLearn************************

1-if you want to add a course , after copying the course to "C:/DigiLearn" you should clean your MetaData.txt by adding the course folder name as it is in the last line and make sure that no empty line is present after it.Possible bug outcome after loging in you get account does not exist message 
2- Use only " .jpg" images while creating courses or you will get all black images in the program.
